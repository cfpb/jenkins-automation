Package: jenkins.automation.builders

[Groovy] Class CheckmarxSecurityJobBuilder

    • jenkins.automation.builders.CheckmarxSecurityJobBuilder

    • Checkmarx Security builder creates a default Checkmarx security build configuration

      emails:
      list of recipients to get notifications
      Parameters:
      name - job name
      description - job description
      scanRepo - a collection of Github repos to scan with Checkmarx
      checkmarxComment - Additional comment(s) to include in the scan results
      useOwnServerCredentials - If set to false then credentials from the Manage Jenkins page are used; serverUrl, username and password parameters are ignored. If set to true then credentials must be specified in serverUrl, username and password parameters
      serverUrl - URL of the Checkmarx server
      username - Checkmarx user name
      password - Checkmarx password
      groupId - Checkmarx group ID passed from environmental variable
      filterPattern - files to exclude
      excludeFolders - folders to exclude
      preset - the Checkmarx preset configuration to use
      vulnerabilityThresholdEnabled - Mark the build as unstable if the number of high severity vulnerabilities is above the specified threshold
      highThreshold - High severity vulnerabilities threshold
      mediumThreshold - Medium severity vulnerabilities threshold
      lowThreshold - Low severity vulnerabilities threshold
      See Also:
      Checkmarx job Example
